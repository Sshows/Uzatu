# ProjectM SIEM/SOAR Defense Slides Outline

## Slide 1. Title

**ProjectM SIEM/SOAR Implementation**  
Purpose: show the project as both a technical implementation and a project management case.

Speaker note: Introduce the project in one sentence and state that the defense covers planning, metrics, testing, and demo evidence.

## Slide 2. Project Goal

- Collect logs from multiple sources
- Detect attacks and correlate events
- Generate alerts for the SOC team

Speaker note: Explain that the system is intended to support operational security monitoring.

## Slide 3. Success Criteria

- Deployment by June 30, 2026
- 45 of 50 sources connected
- Correct rule operation
- Budget within $55,000
- Analysts and admins trained

Speaker note: Emphasize that success is measured, not assumed.
